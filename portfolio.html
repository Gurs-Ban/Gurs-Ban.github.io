UofT Make-a-thon 2020

seeWat

AI vision for the Blind


Made for the 2020 University of Toronto Make-a-thon, this device was made to empower people with visual impairment. seeWat takes a picture from a mounted camera when the user gestures to the ultrasonic sensor and reads the text in the image to the audio outlet of their choice.
This project was made by myself and my 2 friends using a Raspberry Pi 3B+, ultrasonic sensor, and RPi Camera rev 2.0

  • Date: February 2020

N.E.I.R.

NES Emulator In Rust

This project is currently in progress. The goal is to develop an emulator for the Nintendo Entertainmetn system. This project is being worked on by myself and Abhi Agrahari. Currently the Ricoh 2A03 processor is being emulated with my contribution being the MOS Tech 6502 core.
we are coding everyhting in the language Rust, Rust was chosen because it has the unique property of ownership, and lets us develop and display our skills with this relativley new language
